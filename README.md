## Installation

```sh
#Clone GitHub Repo
git clone https://github.com/wfairbanks/purrtect.git

#CD to Project Directory
cd purrtect

#Run Requirements Shell Script
./deps.sh

#Activate Venv
source purrvenv/bin/activate

#Launch App
cd myapp
python3 app.py
```
